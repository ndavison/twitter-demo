# Twitter demo

## setup

This demo requires Python 3 (tested on Python 3.7.5). To install the dependencies:

```
pip install -r requirements.txt
```

## cli

The cli program can be run like so:

```
python tweets.py -u TWITTER_USERNAME
```

See `-h` for more options.
